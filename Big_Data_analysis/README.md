# Big Data Analysis using PySpark

## 📌 Project Overview

This project demonstrates **Big Data Analysis using Apache PySpark** on the **Online Retail Dataset**. The objective is to process a large dataset, perform scalable data analysis, and derive meaningful business insights.

---

## 🎯 Objectives

- Load a large CSV dataset using PySpark
- Clean and preprocess the dataset
- Perform scalable data analysis
- Generate business insights
- Export processed data

---

## 📂 Dataset

**Dataset:** Online Retail Dataset

The dataset contains retail transaction records such as:

- Invoice Number
- Product Description
- Quantity
- Unit Price
- Customer ID
- Country

---

## 🛠 Technologies Used

- Python 3.12
- Apache PySpark 4.1.2
- Pandas
- Jupyter Notebook

---

## 📊 Analysis Performed

- Dataset Exploration
- Missing Value Analysis
- Revenue Calculation
- Top Selling Products
- Country-wise Revenue
- Customer Spending Analysis
- Spark SQL Queries

---

## 📈 Business Insights

- Identified top-selling products.
- Analyzed revenue generated by different countries.
- Identified high-value customers.
- Demonstrated scalable data processing using Apache Spark.

## ▶️ Installation

### Clone Repository

```bash
git clone https://github.com/YourUsername/Big-Data-Analysis-Using-PySpark.git
```

### Move to Project Folder

```bash
cd Big-Data-Analysis-Using-PySpark
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Launch Jupyter Notebook

```bash
jupyter notebook
```

Open **BigDataAnalysis.ipynb** and run all cells.
